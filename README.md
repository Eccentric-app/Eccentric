# Eccentric

## 概要
Eccentricは、学校や日常生活に関する悩みを匿名で投稿できるコミュニティアプリです。  
評価や承認を競うSNSではなく、「安心して言葉にできる場所」を目指しています。

現在、iOS版はすでに公開済みです。

---

## なぜ作ったか
不登校や「学校がつらい」と感じている人は多い一方で、

- 誰にも言えない  
- 言葉にできない  
- 相談するほどでもない  

という状態で止まってしまうケースが多いと感じています。

Eccentricは、  
「相談する前に、少しだけ言葉にできる場所」  
として設計しています。

---

## 現在の状態
- 匿名で投稿できるスレッド型SNS  
- 投稿・削除・通報・ブロック機能あり  
- iOS版はApp Storeで公開済み  
- Android版は開発・公開準備中  

※現在はMVP段階であり、UIや設計には改善余地があります。

---

## 技術スタック
- FlutterFlow（MVP開発）
