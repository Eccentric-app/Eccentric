# Eccentric 技術メモ（エンジニア向け）

## 1. プロジェクト概要
Eccentricは、匿名で悩みを共有できるコミュニティアプリです。  
現在はMVPとしてiOS版を公開済みです。

---

## 2. 技術スタック

### フロントエンド
- FlutterFlow（MVP）
- Flutter / Dart

### バックエンド
- Firebase Authentication
- Cloud Firestore

---

## 3. 現在の状態

- MVPとして動作済み  
- iOS公開済み  
- Android準備中  
- UI・設計は改善余地あり  

---

## 4. アーキテクチャ概要

- フロント：FlutterFlow生成UI  
- バック：Firestoreベース  
- 認証：Firebase Auth  

---

## 5. 課題

### 技術
- FlutterFlow依存  
- コードの可読性低い  
- 状態管理が整理されていない  

### UX
- UIが粗い  
- 投稿導線改善必要  

---

## 6. 今後の方向性

### 短期

### 中期
- 管理画面

### 長期
